# Fotos de Founders — Pendiente

Hasta que se añadan estas fotos, la web muestra "Falta la foto" en la sección de founders.

## Archivos requeridos

| Archivo | Persona | Estado |
|---|---|---|
| `katalin.jpg` | Katalin Savyuk — Founder Estrategia & Producto | ✅ Subida |
| `keoma.jpg` | Keoma Pacheco — Operaciones & Proyectos | ✅ Subida |

## Formato recomendado

- **Dimensiones:** 400 × 500 px (ratio 4:5, orientación vertical/retrato)
- **Formato:** JPG, calidad 85%+
- **Peso máximo:** 200 KB por foto
- **Fondo:** neutro (gris oscuro, negro o exterior difuminado — encaja con el diseño dark)
- **Encuadre:** de hombros hacia arriba, cara centrada

## Cómo añadirlas

1. Preparar la foto con las especificaciones de arriba
2. Guardar como `katalin.jpg` y `keoma.jpg` en esta carpeta (`pagina-web/images/`)
3. Abrir `pagina-web/index.html` en el browser para verificar que se muestran bien
4. Si la cara queda desencuadrada: ajustar `object-position` en el CSS de `.founder-img` en `index.html`
